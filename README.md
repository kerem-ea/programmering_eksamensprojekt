# RSA Nøglegenerator

Et skoleproject der demonstrerer RSA kryptering med en mouse-based seed generator.

## Installation

Kræver Python 3.11 eller nyere.

pip install -r requirements.txt

## Kørsel

Kør programmet fra source/ mappen:

python main.py

## Brug

1. Bevæg musen i den mørke boks for at generere entropi
2. Tryk "GENERER" for at generere RSA nøgler
3. Skriv en besked i inputfeltet
4. Tryk "KRYPTER" for at kryptere og dekryptere beskeden

## Filer

source/main.py - Tkinter applikation
source/rsa.py  - RSA implementering
